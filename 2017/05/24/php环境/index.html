php开发环境搭建

废话不多说，简单易懂的教程来了。

1.先下载XAMPP,一路next。
2.下载phpstorm，激活码去找lanyu的那个，好人一生平安！ http://idea.lanyus.com/。
3.新建php项目，起名字叫index.php，内容如下：


PHP 测试


<?php echo ‘

Hello World

‘; ?>


4.运行的时候会发现报错502，然后编译器右下角，如图。点击超链接。

这个时候你在红框位置，定位到xmapp里面 bin文件下的php文件的位置，顺便记下他的版本


我的是/Applications/XAMPP/xamppfiles/bin/php-7.0.2。
顺路把language level修改成7（这个一定要改！否则就是系统的5版本，网上问题大都出在这里）

5.apply -ok 运行。（over）
